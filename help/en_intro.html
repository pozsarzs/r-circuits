I. Introduction

1. General information
   This software is an R-circuit sizing application.
Supported OSs: FreeBSD, Linux and Windows. Language support: English and Hungarian.

   On BSD and Linux use 'LANG' environment variable to set language of the application. Open Settings window in 'Configuration/Settings' to configure it.
